any other text editor. JAGS is in a diﬀerent category and you probably won’t have seen
it before. JAGS is used to implement Bayesian methods in a straightforward way, and
rjags
allows us to use JAGS from within R. Don’t worry, it’s not too diﬃcult to learn
and use JAGS! We will have a lot of practice using it in the labs.
